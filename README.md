Whac-A-Mole Game 

A simple and fun **Whac-A-Mole** game built in **Java**!  
Test your reflexes and timing skills by whacking as many moles as you can before time runs out.

Features
- Interactive GUI built using **Java Swing/AWT**  
- Random mole appearances to challenge your reaction speed  
- Score tracking and timer mechanism  
- Lightweight and easy to run on any Java-supported system  

How to Run
 Clone this repository
